# Streamlit ToDo Application

A fast and lightweight task management application built with Python and the Streamlit framework.

## Project Structure

- `demo.py`: The single-script implementation of the ToDo interface and logic.
- `requirements.txt`: Python package dependencies.

## Key Features
- **Instant Deployment**: Minimal code required for a fully functional web interface.
- **Interactive UI**: Real-time task filtering and management directly in the browser.
- **Persistent State**: Efficiently handles task state during user sessions.

## Getting Started

1.  **Install Streamlit**:
    ```bash
    pip install streamlit
    ```
2.  **Run the App**:
    ```bash
    streamlit run demo.py
    ```
